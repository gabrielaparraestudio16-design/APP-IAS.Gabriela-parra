
# Detector de Emociones con DeepFace

## Instalación

1. Instala Python 3.10+
2. Abre CMD en esta carpeta
3. Ejecuta:

```
pip install -r requirements.txt
```

## Ejecución

```
python main.py
```

## Uso

1. Cargar imagen
2. Analizar emoción
3. Ver resultados en pantalla

## Notas

- Compatible con TensorFlow 2.18.0
- Si no detecta rostro, se mostrará un error

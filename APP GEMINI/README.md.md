# EMO-SCAN: Detector de Emociones con DeepFace

Esta aplicación permite cargar imágenes estáticas y analizar las emociones faciales presentes utilizando Redes Neuronales Profundas.

## Requisitos Previos
* Python 3.9 o superior.
* Conexión a Internet (solo la primera vez para descargar los modelos pre-entrenados).

## Instalación Paso a Paso

1. **Clonar o descargar** este repositorio en una carpeta local.
2. **Abrir la terminal (CMD)** en esa carpeta.

# 🎭 EmotionVision - Detector de Emociones Faciales

[![Python Version](https://img.shields.io/badge/python-3.8%20%7C%203.9%20%7C%203.10%20%7C%203.11-blue)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)
[![DeepFace](https://img.shields.io/badge/DeepFace-0.0.79-orange)](https://github.com/serengil/deepface)
[![TensorFlow](https://img.shields.io/badge/TensorFlow-2.15.0-red)](https://www.tensorflow.org/)

Una aplicación de escritorio moderna que utiliza inteligencia artificial para detectar y analizar emociones faciales en imágenes estáticas.

## ✨ Características

- 🎯 **Detección precisa**: Identifica 7 emociones diferentes
- 🖼️ **Interfaz intuitiva**: Diseño moderno y fácil de usar
- 📊 **Resultados detallados**: Porcentaje de confianza para cada emoción
- 🔍 **Visualización**: Muestra el rostro detectado con rectángulo delimitador
- ⚡ **Procesamiento en hilos**: No bloquea la interfaz durante el análisis
- 🌍 **Soporte multilingüe**: Resultados en español

## 🎨 Emociones Detectadas

| Emoción | Descripción |
|---------|-------------|
| 😊 FELIZ | Alegría, felicidad |
| 😢 TRISTE | Tristeza, melancolía |
| 😠 ENFADADO | Enojo, ira |
| 😲 SORPRESA | Asombro, sorpresa |
| 😨 MIEDO | Temor, miedo |
| 🤢 ASCO | Disgusto, repulsión |
| 😐 NEUTRAL | Estado neutral |

## 📋 Requisitos del Sistema

- **Python**: 3.8, 3.9, 3.10 o 3.11 (NO 3.12 o superior)
- **Sistema Operativo**: Windows 10/11, macOS, Linux
- **RAM**: Mínimo 4GB (recomendado 8GB)
- **Espacio en disco**: ~500MB para los modelos de IA
- **Conexión a internet**: Necesaria para la primera instalación
